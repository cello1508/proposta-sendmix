Necessário

O que fica por sua conta

Para os agentes funcionarem, precisamos de acesso às suas plataformas. Todas as APIs e tokens são do cliente.

Meta Business / Ads Manager

Acesso à conta de anúncios + token de API (System User). Nós orientamos a criação.

Google Ads

Acesso à conta Google Ads + credenciais OAuth e developer token. Tudo no seu nome.

TikTok Ads

Acesso à conta TikTok for Business + token de API. Nós guiamos o processo de aprovação.

Trello

Conta Trello com board dedicado para os agentes. API key e token do seu workspace.

Importante: Todas as credenciais, tokens e acessos de API ficam 100% no nome do cliente. A MGTInc configura e orienta, mas a propriedade é sua. Custos de API das plataformas (quando houver) são por conta do cliente.
